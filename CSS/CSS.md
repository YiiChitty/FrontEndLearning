## 表格布局的弊端

- 可访问性查，内容从左到右从上到下的读取并不是总有意义，无障碍工具从文档中获取的数据会混乱 <br/>
- 难以实现响应式 <br/>
- 可维护性差 <br/>
- 不够语义化 <br/>
- 加载速度慢，嵌套表多 <br/>
